# 基于深度学习的音乐推荐系统 python1065

## 项目概述


## 技术栈

- **开发语言**：Python
- **架构设计**：B/S框架
- **前端技术**：HTML
- **数据库**：MySQL
- **核心算法**：深度学习

## 功能模块

- **用户登录**：通过登录系统，用户可以实现个性化推荐服务。
- **音乐标签设定**：通过为音乐添加标签，便于同类型音乐的推荐。
- **搜索与推荐**：用户可通过歌名或关键词搜索，系统展示相似度高和相关性强的歌曲推荐。
- **推荐排名**：系统根据推荐算法，为用户推荐相关性高的前十大歌曲。

## 系统角色

- **用户**：通过登录使用系统，接收音乐推荐。
- **系统**：分析用户数据，提供个性化音乐推荐。

1. 环境配置：准备Python运行环境，安装必要的库。
